# VAE vs GAN 对比实验

## 概述

本项目从零实现了两种经典的生成模型——**变分自编码器 (VAE)** 和**生成对抗网络 (GAN)**，并在相同条件下进行对比实验，帮助理解两者的原理和区别。

## 核心原理

### VAE (Variational Autoencoder)

VAE 通过学习数据的潜在分布来生成新样本。核心是最大化证据下界 (ELBO):

```
ELBO = E_q[log p(x|z)] - KL(q(z|x) || p(z))
     = 重构损失         + KL 散度正则化
```

**关键组件:**
- **编码器**: 将输入映射为潜在分布参数 (μ, σ²)
- **重参数化技巧**: z = μ + σ·ε, ε~N(0,1)，使采样可微分
- **解码器**: 从潜在变量重构输入

### GAN (Generative Adversarial Network)

GAN 通过两个网络的对抗博弈来学习生成:

```
min_G max_D  E[log D(x)] + E[log(1 - D(G(z)))]
```

**关键组件:**
- **生成器 (G)**: 从随机噪声生成假样本
- **判别器 (D)**: 区分真实样本和假样本
- **对抗训练**: 交替优化 G 和 D

## 对比总结

| 维度 | VAE | GAN |
|------|-----|-----|
| 训练目标 | 最大化 ELBO | 极小极大博弈 |
| 训练稳定性 | ✅ 稳定 | ❌ 不稳定 |
| 生成清晰度 | ❌ 较模糊 | ✅ 更清晰 |
| 模式崩塌 | ✅ 不容易 | ❌ 容易发生 |
| 似然估计 | ✅ 可以计算 | ❌ 无法直接计算 |
| 潜在空间 | ✅ 连续有结构 | ❌ 无显式结构 |
| 典型应用 | 数据压缩/异常检测 | 图像生成/超分辨率 |

## 实现细节

### VAE 训练技巧
- 使用 log σ² 而非 σ²，保证数值稳定
- KL 散度有解析解（两个高斯分布）
- 重参数化技巧使采样操作可微分

### GAN 训练技巧
- Adam 优化器 β1=0.5（减少动量，稳定训练）
- 生成器使用 BatchNorm，判别器使用 Dropout
- 非饱和损失（替代原始的 minimax 损失）
- 判别器使用 LeakyReLU 避免梯度消失

## 选择建议

- **需要稳定训练和潜在空间结构** → 选 VAE
- **需要高质量图像生成** → 选 GAN
- **两者都要** → 考虑 VAE-GAN 混合模型
